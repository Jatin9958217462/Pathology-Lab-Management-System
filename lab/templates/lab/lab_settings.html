🖥️ Multi-Device / Multi-Computer Setup Guide
💾
Pendrive Mein Data
1. Pendrive lagao (jaise E:)
2. Database path set karo:
E:\pathlab\db.sqlite3
3. Save karke server restart karo
4. Ab kisi bhi PC mein pendrive lagao — data wahi rahega
🌐
Ek Server, Baaki Browser Se
1. Ek computer pe PathLab chalao
2. start.bat se 0.0.0.0:8000 pe chalega
3. Dusre computers browser mein kholein:
http://[server-ip]:8000
4. Sabka data ek hi database mein
📁
Network Folder / NAS
1. Shared folder ka path set karo:
\\192.168.1.5\share\db.sqlite3
2. Sabhi computers same path use karein
3. ⚠️ Ek saath sirf ek hi user write kare — SQLite limitation
